Telegram Commands (Two-Way)

Send these commands to your Telegram bot:

  • /status - Get system status
  • /summary - Today's summary
  • /cameras - Camera status
  • /count - Entry/exit count
  • /alerts - Recent alerts
  • /faces - Face stats
  • /plates - Recent plates
  • /report - Generate daily report now
  • /help - Show all commands

  • Add Known Face: Faces auto-save → go to Faces page to rename

  • Blacklist Person: Faces page → click Blacklist
  • Blacklist Vehicle: Plates page → click Ban
